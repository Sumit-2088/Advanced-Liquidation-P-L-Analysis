# 💸 Advanced Liquidation P&L Analysis

[Live Website 🚀](https://jangid.onrender.com)  
[View Demo 📽️](#coming-soon)

A powerful web-based tool to calculate **profit & loss from crypto liquidations**. Designed for traders who want quick and accurate insights into their positions.

---

## 🧮 Features

- ✅ Real-time P&L calculation
- 📉 Entry price, Exit price, and Leverage inputs
- 📊 Liquidation price estimation
- 🔢 Support for long & short positions
- ⚡ Lightning-fast and easy to use
- 🖥️ Fully responsive UI
